Přehled JSON struktury

JSON Schéma zachycující strukturu pro Kategorie orgánů veřejné moci je definováno v souboru ./kategorie-orgánů-veřejné-moci.schema.json. Datová sada je tvořena seznamem prvků odpovídajících datové struktuře Kategorie orgánů veřejné moci. Prvky jsou uvedeny v poli `položky`.

  • Kategorie orgánů veřejné moci
    • identifikátor: povinná (1 - 1) položka typu Řetězec dle regulárního výrazu ^KO[0-9]+$
    • název: povinná (1 - 1) položka typu Text
    • ustanovení-definice-kategorie-esbírka: nepovinná (0 - ∞) položka typu IRI (Ustanovení právního předpisu)
    • ustanovení-definice-kategorie-ostatní: nepovinná (0 - ∞) položka typu Ustanovení právního předpisu
    • ustanovení-složení-kategorie-esbírka: nepovinná (0 - ∞) položka typu IRI (Ustanovení právního předpisu)
    • ustanovení-složení-kategorie-ostatní: nepovinná (0 - ∞) položka typu Ustanovení právního předpisu
    • podřazené-kategorie: nepovinná (0 - ∞) položka typu Zařazení kategorie orgánů veřejné moci
    • ohlašovatel: povinná (1 - 1) položka typu IRI (Ústřední správní úřad)
    • platnost-od: povinná (1 - 1) položka typu Datum
    • platnost-do: nepovinná (0 - 1) položka typu Datum
  • Zařazení kategorie orgánů veřejné moci
    • kategorie: povinná (1 - 1) položka typu IRI (Kategorie orgánů veřejné moci)
    • zařazení-od: povinná (1 - 1) položka typu Datum
    • zařazení-do: nepovinná (0 - 1) položka typu Datum
