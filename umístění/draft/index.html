Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování umístění v poskytovaných datech. Tato norma je navržena jako sdílená norma pro použití v dalších, specifických otevřených formálních normách. Norma obsahuje schémata a příklady dat v datovém modelu RDF [[rdf11-concepts]] a ve formátech CSV [[rfc4180]], JSON [[ECMA-404]] a XML [[xml]].

Specifikace

V této sekci jsou definovány jednotlivé položky, které lze použít k určení umístění v poskytovaných datech. Umístění může být specifikováno jako adresa, globálně identifikovaný objekt a nebo jako geometrický útvar.

Tato otevřená formální norma strukturuje umístění v souladu s ISA2 Core Location Vocabulary.

Umístění

Umístění může být specifikováno jako adresa, globálně identifikovaný objekt a nebo jako geometrický útvar. Příkladem umístění může být louka, na které se koná festival, adresa sídla společnosti, či prodaná parcela na Měsíci. Tato třída odpovídá třídě Location z ISA2 Core Location Vocabulary a Dublin Core Metadata Initiative.

Příklady

V této sekci jsou uvedeny příklady reprezentace umístění v různých formátech a v různých úrovních detailu. Datová schémata, která jsou použita v příkladech, lze přímo použít a zahrnout do publikačních plánů, případně rozšířit dle vlastních potřeb. Pro formáty JSON [[!ECMA-404]] a XML [[!xml]] jsou k dispozici příslušná schémata, tedy Schéma pro formát JSON v jazyce JSON Schema [[json-schema]] a Schéma pro formát XML v jazyce XML Schema [[xmlschema11-1]]. Lze také použít zápis v CSV. Soubor v CSV [[!rfc4180]] je nutné doplnit schématem dle specifikace CSV on the Web [[tabular-data-primer]].

Pokud je umístění použito v rámci jiného datového standardu, tak se jména sloupců v CSV tabulce tvoří názvy atributů. Zanořování je řešeno spojením názvu entity z jiného standardu, například lokace, a z názvu atributu adresy jejich spojením pomocí znaku podtržítka _, například lokace_geometrie.

Specifikace umístění identifikátorem

Jedná se o nejčistší způsob určení umístění pomocí IRI příslušného geografického objektu.

Stáhnout JSON, Stáhnout JSON schéma 

{
    "iri": "http://publications.europa.eu/resource/authority/continent/ANTARCTICA"
}

Stáhnout XML, Stáhnout XML schéma 

<?xml version="1.0" encoding="UTF-8" ?>
<umístění
    xmlns="https://data.gov.cz/otevřené-formální-normy/umístění/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/umístění/ 
                        https://data.gov.cz/otev%C5%99en%C3%A9-form%C3%A1ln%C3%AD-normy/um%C3%ADst%C4%9Bn%C3%AD/draft/sch%C3%A9mata/um%C3%ADst%C4%9Bn%C3%AD.xsd">
    <iri>http://publications.europa.eu/resource/authority/continent/ANTARCTICA</iri>
</umístění>

Stáhnout CSV 

iri
http://publications.europa.eu/resource/authority/continent/ANTARCTICA

K souboru CSV je třeba připojit metadata dle standardu CSV on the Web [[tabular-data-primer]]. Například pokud se CSV soubor jmenuje 0.csv, je s ním publikován i metadatový soubor 0.csv-metadata.json:

Stáhnout deskriptor 

{
    "@context": [
        "http://www.w3.org/ns/csvw",
        {"@language": "cs"}
    ],
    "url": "0.csv",
    "header": true,
    "tableSchema": {
        "columns": [
            {
                "titles": "iri",
                "dc:description": "Geografický identifikátor",
                "datatype": "anyURI",
                "required": true
            }
        ]
    }
}

Specifikace umístění názvem či geometrií

Pokud je třeba specifikovat umístění názvem nebo geometrií, lze to provést tak jako v následujících příkladech.

Stáhnout JSON, Stáhnout JSON schéma 

{
    "název": "Národní park Šumava",
    "geometrie": "POINT(48.9720274 13.6309462)"
}

Stáhnout XML, Stáhnout XML schéma 

<?xml version="1.0" encoding="UTF-8" ?>
<umístění
    xmlns="https://data.gov.cz/otevřené-formální-normy/umístění/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/umístění/ 
                        https://data.gov.cz/otev%C5%99en%C3%A9-form%C3%A1ln%C3%AD-normy/um%C3%ADst%C4%9Bn%C3%AD/draft/sch%C3%A9mata/um%C3%ADst%C4%9Bn%C3%AD.xsd">
    <název>Národní park Šumava</název>
    <geometrie>POINT(48.9720274 13.6309462)</geometrie>
</umístění>

Stáhnout CSV 

název,geometrie
Národní park Šumava,POINT(48.9720274 13.6309462)

K souboru CSV je opět potřeba připojit metadatový popsiek dle standardu CSV on the Web [[!tabular-data-primer]]:

Stáhnout deskriptor 

{
    "@context": [
        "http://www.w3.org/ns/csvw",
        {"@language": "cs"}
    ],
    "url": "1.csv",
    "header": true,
    "tableSchema": {
        "columns": [
            {
                "name": "nazev",
                "titles": "název",
                "dc:description": "Geografický název",
                "datatype": "string"
            }, {
                "titles": "geometrie",
                "dc:description": "Geometrie",
                "datatype": "string"
            }
        ]
    }
}

Umístění formou adresy

Umístění může být specifikováno adresou. Oproti přímému použití adresy bez mezikroku v podobě Umístění lze specifikovat dodatečná data týkající se umístění, například název.

Stáhnout JSON, Stáhnout JSON schéma 

{
    "název": "Centrotex",
    "adresa": {
        "číslo_domovní": 3,
        "název_ulice": "nám. Hrdinů",
        "název_obce": "Praha",
        "psč": "14021"
    }
}

Stáhnout XML, Stáhnout XML schéma 

<?xml version="1.0" encoding="UTF-8" ?>
<umístění
    xmlns="https://data.gov.cz/otevřené-formální-normy/umístění/"
    xmlns:adresa="https://data.gov.cz/otevřené-formální-normy/adresy/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="https://data.gov.cz/otevřené-formální-normy/umístění/ 
                        https://data.gov.cz/otev%C5%99en%C3%A9-form%C3%A1ln%C3%AD-normy/um%C3%ADst%C4%9Bn%C3%AD/draft/sch%C3%A9mata/um%C3%ADst%C4%9Bn%C3%AD.xsd">
    <název>Centrotex</název>
    <adresa>
        <adresa:číslo_domovní>3</adresa:číslo_domovní>
        <adresa:název_obce>Praha</adresa:název_obce>
        <adresa:název_ulice>nám. Hrdinů</adresa:název_ulice>
        <adresa:psč>14021</adresa:psč>
    </adresa>
</umístění>

Stáhnout CSV 

název,adresa_číslo_domovní,adresa_název_ulice,adresa_název_obce,adresa_psč
Centrotex,3,nám. Hrdinů,Praha,14021
