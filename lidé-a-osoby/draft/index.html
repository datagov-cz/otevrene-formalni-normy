Abstrakt

Tento dokument je otevřenou formální normou ve smyslu § 3 odst. 9 zákona č. 106/1999 Sb., o svobodném přístupu k informacím, pro zveřejňování informací o lidech a osobách. Je navržena jako sdílená norma pro použití v dalších specifických otevřených formálních normách. Ukazuje, jak reprezentovat informace o jménech lidí. Norma je ilustrována příklady dat ve formátech CSV [[rfc4180]], JSON [[ECMA-404]], XML [[xml]] a RDF [[rdf11-concepts]].

Specifikace

V této sekci jsou definovány vlastnosti lidí a osob. Pro každou vlastnost je uveden její identifikátor, který je pro její reprezentaci použit ve všech datových formátech, její název, datový typ, popis a příklad.

Příklady

V této sekci jsou uvedeny příklady reprezentace lidí. Datová schémata, která jsou použita v příkladech, lze přímo použít a zahrnout do publikačních plánů, případně rozšířit dle vlastních potřeb. Pro formáty JSON a XML jsou k dispozici příslušná schémata, tedy Schéma pro formát JSON v jazyce JSON Schema [[!json-schema]] a Schéma pro formát XML v jazyce XML Schema [[!xmlschema11-1]]. Lze také použít zápis v CSV. Soubor v CSV je nutné doplnit schématem dle specifikace CSV on the Web [[!tabular-data-primer]].

Pokud je kontakt použit v rámci jiného datového standardu, tak se jména sloupců v CSV tabulce tvoří názvy atributů. Zanořování je řešeno spojením názvu entity z jiného standardu, například kontaktní_osoba a z názvu atributu člověka jejich spojením pomocí znaku podtržítka _, například kontaktní_osoba_příjmení.

Úplný příklad jména člověka.

Stáhnout JSON 

{
    "jméno": "Jakub",
    "příjmení": "Novák",
    "tituly_před": "RNDr.",
    "tituly_za": "Ph.D.",
    "text": "RNDr. Jakub Novák, Ph.D."  
}

Stáhnout XML 

<?xml version="1.0" encoding="UTF-8"?>
<člověk:člověk xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
xmlns:člověk="https://ofn.gov.cz/lidé-a-osoby/" 
xsi:schemaLocation="https://ofn.gov.cz/lidé-a-osoby/ https://ofn.gov.cz/lid%C3%A9-a-osoby/draft/sch%C3%A9mata/%C4%8Dlov%C4%9Bk-%C4%8Di-osoba.xsd">
    <člověk:jméno>Jakub</člověk:jméno>
    <člověk:příjmení>Novák</člověk:příjmení>
    <člověk:tituly_před>RNDr.</člověk:tituly_před>
    <člověk:tituly_za>Ph.D.</člověk:tituly_za>
    <člověk:text>RNDr. Jakub Novák, Ph.D.</člověk:text>
</člověk:člověk>

Stáhnout CSV 

jméno,příjmení,tituly_před,tituly_za,text
Jakub,Novák,RNDr.,Ph.D.,"RNDr. Jakub Novák, Ph.D."

Člověk s přidaným vlastním atributem

Do schématu si lze v případě potřeby přidávat další atributy. V následujících příkladech jsou nad rámec standardu přidán atribut poznámka.

Stáhnout JSON: 

{
    "jméno": "Jakub",
    "příjmení": "Novák",
    "tituly_před": "RNDr.",
    "tituly_za": "Ph.D.",
    "text": "RNDr. Jakub Novák, Ph.D.",
    "poznámka": "Kamarád"
}

V XML se vlastní atributy přidávají na závěr kontaktu, do vlastního XML jmenného prostoru (XML namespace [[!xml-names]]). V tomto příkladu jsou nové atributy přidány do XML jmenného prostoru x.

Stáhnout XML: 

<?xml version="1.0" encoding="UTF-8"?>
<člověk:člověk xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
xmlns:x="http://příklad.cz" 
xmlns:člověk="https://ofn.gov.cz/lidé-a-osoby/" 
xsi:schemaLocation="https://ofn.gov.cz/lidé-a-osoby/ https://ofn.gov.cz/lid%C3%A9-a-osoby/draft/sch%C3%A9mata/%C4%8Dlov%C4%9Bk-%C4%8Di-osoba.xsd">
    <člověk:jméno>Jakub</člověk:jméno>
    <člověk:příjmení>Novák</člověk:příjmení>
    <člověk:tituly_před>RNDr.</člověk:tituly_před>
    <člověk:tituly_za>Ph.D.</člověk:tituly_za>
    <člověk:text>RNDr. Jakub Novák, Ph.D.</člověk:text>
    <x:poznámka>Kamarád</x:poznámka>
</člověk:člověk>

Alternativně je možné si nadefinovat celé nové XML schéma, které by vycházelo ze zde uvedeného základního XML schématu.

Jakub,Novák,RNDr.,Ph.D.,"RNDr. Jakub Novák, Ph.D.","Kamarád"
